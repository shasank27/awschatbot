# AWS Chatbot

A chatbot project designed to interact with AWS services with Whatsapp. This repository contains the source code, configuration, and scripts required to deploy and run the AWS Chatbot.

## Project Structure

```
awschatbot/
├── README.md
├── lambda_functions.py
├── agent.py
├── tasks.md
├── docker/
│   ├── Dockerfile
│   └── requirements.txt
└── assests/
    ├── sequenceDiagram
    └── architecture.png
```

## Getting Started

### Prerequisites

- Python 3.8+
- AWS CLI configured
- pip
